API Documentation

Product List API

This API allows to get List of Products which admin is created. Token Require in this.

TOKEN KEY: x-access-token

Endpoint: product/list

Method: GET

cURL :- curl --location 'http://localhost:3000/product/list'

API Response

{
     "status": true,
     "message": "Product List",
     "data": {
        // Product Data
     }
}

Add Product API

This API allows Admin to add/create a Product for the Users.

Endpoint: product/create

Method: POST

TOKEN KEY: x-access-token

Accepted Keys:

  • name
  • price
  • color:In Array
  • size:In Array
  • description
  • quantity
  • isExchangeable
  • isRefundable
  • isRepairable
  • productImages

Note: By default type is user

cURL :- curl --location 'http://localhost:3000/product/create' \ --header 'x-access-token: //Token' \ --form 'name=" test_pro"' \ --form 'price=" 500"' \ --form 'color[]=" blue"' \ --form 'size[]=" M"' \ --form 'description=" This is a test Product"' \ --form 'quantity=" 5"' \ --form 'isExchangeable="true"' \ --form 'isRefundable="true"' \ --form 'productImages=@"/home/nodejs/Downloads/sideImage.png"' \ --form 'category="66a8e9817eb334de85fe4b5c"'

API Payload

{
    name: // product name
    price: // product price
    color[]: // product color in array
    size[]: // product size in array
    description: // product description
    quantity: // product quantity
    isExchangeable: // Boolean // default is true
    isRefundable: // Boolean // default is true
    productImages: // product images in array // Buffer image
    category: // category object id
    // other keys to Add
}

API Response

{
    "status": true,
    "message": "Product Created Successfully",
    "data": // product data
}

Edit Product API

his API allows to edit a specific product. Token Require in this. Product id pass in body of API

Endpoint: /product/edit

Method: PUT

TOKEN KEY: x-access-token

Accepted Keys:

  • name
  • price
  • color[]
  • size[]
  • quantity
  • isExchangeable
  • isRefundable
  • productImages
  • description

Note: By default type is user

cURL :- curl --location --request PUT 'http://localhost:6000/product/edit' \ --header 'x-access-token: //token \ --form 'name=" test_pro"' \ --form 'price=" 500"' \ --form 'color[]=" blue"' \ --form 'size[]=" M"' \ --form 'description=" This is a test Product"' \ --form 'quantity=" 5"' \ --form 'isExchangeable="true"' \ --form 'isRefundable="true"' \ --form 'productImages=@"/path/to/file"'

API Payload

{

    size[]: // product size in array

    quantity: // product quantity
    isExchangeable: // Boolean // default is true
    isRefundable: // Boolean // default is true
    productImages: // product images in array // Buffer image
    // other keys to Add
}

API Response

{
    "status": true,
    "message": "Product Updated Successfully",
    "data": // product Data
}

Delete Product API

This API allows Admin to Delete a Product by passing product id in query params.

Endpoint: /product/delete

Method: Delete

TOKEN KEY: x-access-token

cURL :- curl --location --request DELETE 'http://localhost:3000/product/delete?product_id=6698c70217f8a2dbe05e7115' \ --header 'x-access-token: //Token'

API Response

{
    "status": true,
    "message": "Product Deleted Successfully",  
}
