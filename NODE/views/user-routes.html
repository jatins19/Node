API Documentation

profile API

This API allows to get user details. Token Require in this.

TOKEN KEY: x-access-token

Endpoint: /profile

Method: GET

cURL :- curl --location --request GET 'http://192.168.1.82:3000/profile' \ --form 'email="anm@yopmail.com"' \ --form 'profile_pic=@"/home/admin1212/Pictures/Screenshots/Screenshot from 2024-07-16 16-47-27.png"'

API Response

{
     "status": true,
     "message": "Profile Info",
     "data": {
        // user Data
     }
}

Edit Profile API

This API allows users to update there profile.

Endpoint: /profile/edit

Method: PUT

Accepted Keys:

  • email
  • first_name
  • last_name
  • dob
  • address
  • phone_number
  • country_code
  • username :"if you have user name"
  • profile_pic :"if you want to add or update profile image"

cURL :- curl --location --request PUT 'http://localhost:3000/profile/edit' \ --form 'email="anm@yopmail.com"' \ --form 'profile_pic=@"/home/admin1212/Pictures/Screenshots/Screenshot from 2024-07-16 16-47-27.png"'

API Payload

{
    "email": "user@example.com",  
    "profile_pic": "image"
    // other keys to update
}

API Response

{
    "status": true,
    "message": "Updated successfully",
    "data": {
        // user Data
    }
}

change Password API

This API allows users to Password from profile dashbord

Endpoint: /profile/changePassword

Method: PUT

cURL :- curl --location --request PUT 'http://localhost:3000/profile/changePassword' \ --header 'x-access-token;' \ --header 'Content-Type: application/json' \ --data '{ "currentPassword": "12345678", "newPassword": "1234567890" }'

API Payload

{
    "currentPassword": "12345678",
    "newPassword": "12345678"
}

API Response

{
    "status": true,
    "message": "Password Change Sucessfully."
}
