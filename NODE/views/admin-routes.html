Plan List API

This API allows to get plan details. Token Require in this. Type is use for getting different kind of subscription plans.

Endpoint: superadmin/list/:type

Method: GET

cURL :- curl --location 'http://localhost:3000/superadmin/list/user'

API Response

{
     "status": true,
     "message": "Plan List",
     "data": {
        // Plan Data
     }
}

Add Plan API

This API allows Super Admin to add/create a Subscription Plan for the Users.

Endpoint: superadmin/add-subscription

Method: POST

TOKEN KEY: x-access-token

Accepted Keys:

  • plan_name
  • duration : it should be in number's of month
  • price
  • type : it is set according to the plan usage(user, product etc.)

Note: By default type is user

cURL :- curl --location 'http://localhost:3000/superadmin/add-subscription' \ --header 'x-access-token;' \ --header 'Content-Type: application/json' \ --data '{ "plan_name":"Basic", "duration":2, "price":2500, "type":"user" }'

API Payload

{
    "plan_name": "basic",  
    "duration": 2 //months,
    "price": "2500" //amount,
    "type": "user"
    // other keys to update
}

API Response

{
    "status": true,
    "message": "Plan Creates Successfully",
    "data": // plan_id
}

Create Admin API

This API allows Super Admin to add/create a Admin.

Endpoint: superadmin/create-admin

Method: POST

TOKEN KEY: x-access-token

Accepted Keys:

  • email
  • password
  • first_name
  • last_name
  • dob
  • address
  • phone_number
  • country_code

cURL :- curl --location 'http://localhost:3000/superadmin/create-admin' \ --header 'Content-Type: application/json' \ --data-raw '{ "first_name":"Dheeraj", "last_name":"gehlot", "dob":"2220-10-08T00:00:00.000+00:00", "address":"vizag", "email":"d@gmail.com", "phone_number":"1651561", "country_code":"+1", "password":"12345678" }'

API Payload

{
    "first_name":"user",
    "last_name":"name",
    "dob":"2220-10-08T00:00:00.000+00:00",
    "address":"USA",
    "email":"example@gmail.com",
    "phone_number":"1651561",
    "country_code":"+1",
    "password":"12345678"
}

API Response

{
    "status": true,
    "message": "Admin Added Successfull", 
}
